# Code of Conduct

## Our pledge
We want a welcoming, inclusive, and respectful community for everyone.

## Expected behavior
- Be respectful and considerate.
- Assume good intent and be constructive.
- Focus on what is best for the project and community.

## Unacceptable behavior
- Harassment, discrimination, or abusive language.
- Trolling, personal attacks, or sustained disruption.
- Sharing private information without consent.

## Enforcement
Project maintainers may remove or edit contributions that violate this code of conduct.

## Reporting
If you experience or witness unacceptable behavior, contact the maintainer at contact@oguzhanumutlu.com.
